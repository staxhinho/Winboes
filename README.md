<details>
  <summary><h2>⚙️ Running locally for development</h2></summary>

Start cloning Konkord:

```
git clone https://github.com/staxhinho/Winboes.git
```


Next, install its dependencies (make sure pnpm is installed on your machine):

```
pnpm install
```

To start, run:

```
pnpm start
```

To build the app:
```
pnpm run make
```

</details>
